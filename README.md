# handlevakten

Denne er en webapplikasjon som jeg utvikler i emnet DAT130
Webprogrammering og Databaser. Prosjektet er i en tidlig utviklingsfase og implementeres
med Flask og MySQL på backend, samt HTML, CSS og JavaScript på frontend. Applikasjonen
skal la brukere søke, filtrere og sammenligne priser på dagligvarer ved hjelp av kassal.app
API, samt opprette konto og lagre ønskeliste
